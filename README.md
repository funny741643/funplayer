# 项目启动
* npm i
* npm run dev:funplayer

# funplayer基础配置项

| __属性__           | __描述__                               |
| ------------------ | -------------------------------------- |
| id/el              | 播放器实例占位dom                      |
| url                | 播放源                                 |
| width/height       | 播放器尺寸                             |
| volume             | 预设音量大小                           |
| autoplay           | 自动播放                               |
| loop               | 循环播放                               |
| poster             | 视频封面图                             |
| playbackRateUnit   | 倍速展示单位                           |
| isShowCoverPreview | 是否开启预览                           |
| playNext           | 视频跳转播放下一集，需提供视频资源列表 |
| download           | 设置download控件显示                   |
| screenShot         | 截屏配置                               |


# funplayer内置控件

| __名称__       | __描述__   |
| -------------- | ---------- |
| process        | 进度条     |
| play           | 播放按钮   |
| volume         | 音量调节   |
| fullscreen     | 全屏       |
| web-fullscreen | 网页全屏   |
| time           | 播放时间   |
| i18n           | 多语种切换 |
| replay         | 播放提示   |
| loading        | 加载提示   |
| settings       | 设置       |
| error          | 错误提示   |
